Hobi

Bermain Game

Saya suka meluangkan waktu untuk bermain game karena aktivitas ini memberikan kesempatan bagiku untuk bersantai dan menghibur diri. Selain itu, saya juga dapat melupakan sejenak segala tekanan dan kesibukan yang sedang dihadapi. Melalui permainan, saya bisa melatih konsentrasi, ketangkasan, dan kemampuan strategis saya.

Mendengarkan musik

Saya menikmati alunan musik sebagai hobi santai yang sering saya lakukan untuk merelaksasi pikiran dan menemani waktu luang. Musik memiliki kekuatan untuk mengubah suasana hati saya menjadi lebih positif dan membawa kedamaian.

Programming

Saya memiliki hobi dalam programming karena kegiatan ini memungkinkan saya untuk mengeksplorasi dan berkembang dalam dunia teknologi informasi. Melalui kegiatan ini, saya dapat meningkatkan keterampilan pemrograman saya, serta mengeksplorasi berbagai teknologi baru. Selain itu, programming juga dapat saya gunakan sebagai sarana pengembangan diri dan kreativitas.

Belajar Server Linux

Saya tertarik menyelami dunia server Linux serta belajar self-hosting sebagai hobi yang memperluas pengetahuan saya tentang sistem operasi Linux dan pengelolaan server. Melalui kegiatan ini, saya bisa memahami lebih dalam tentang infrastruktur teknologi informasi, serta meningkatkan keterampilan saya dalam mengelola server sendiri
